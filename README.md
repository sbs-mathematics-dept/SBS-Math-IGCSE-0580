# SBS Interactive Mathematics
## Cambridge IGCSE 0580 Extended

A free, open-access interactive mathematics website for students at **SBS International School Chiangmai**.

### 🌐 Live site
Once pushed to GitHub Pages, this site will be available at:
`https://[your-username].github.io/sbs-math/`

---

---

### 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `sbs-math`)
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages → Source → Deploy from branch → main**
4. Your site will be live within a few minutes

### Or via Git:
```bash
git init
git add .
git commit -m "Initial SBS Interactive Mathematics site"
git remote add origin https://github.com/[username]/sbs-math.git
git push -u origin main
```

Then enable GitHub Pages in repository Settings.

### ⚙️ Technical notes
- **Pure HTML/CSS/JavaScript** — no frameworks, no build step, no server needed
- All labs work **offline** once loaded
- **Mobile-friendly** — responsive design, touch support in circle theorems lab
- **No login required** — students access everything instantly
- Cambridge IGCSE 0580 Extended syllabus aligned
